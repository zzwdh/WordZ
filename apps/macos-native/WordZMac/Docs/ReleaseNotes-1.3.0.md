# WordZ 1.3.0 Release Notes

> Draft
>
> 这是一份 `1.3.0` 的预备版本说明骨架，随着开发推进逐步填充。

## 概览

WordZ 1.3.0 聚焦于“研究工作流与可复现实验”。这次更新的重点不是单独添加某一个分析页，而是把语料筛选、参考语料、分析参数、结果导出和方法说明真正串成一条研究闭环。

## 计划中的主要更新

### 1. 参考语料与子语料工作流

- [ ] 子语料构建器
- [ ] 命名语料集
- [ ] 固定参考语料与自动参考策略增强
- [ ] Compare 导出附带更完整方法摘要

### 2. 元数据驱动的语料库管理

- [ ] 语料库筛选器
- [ ] 批量元数据编辑
- [ ] 元数据完整性提示
- [ ] 主工作区直接选择命名语料集

### 3. 大语料性能专项

- [ ] 大文件流式导入
- [ ] 导入取消与进度反馈
- [ ] `.db` 索引增强
- [ ] 高频分析路径缓存细化

### 4. 可复现分析预设

- [ ] 分析预设保存与复用
- [ ] 方法说明面板
